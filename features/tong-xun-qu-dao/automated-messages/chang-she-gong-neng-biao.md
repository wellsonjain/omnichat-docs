# 常設功能表

![](../../../.gitbook/assets/常設功能表.png) ![](../../../.gitbook/assets/IMG\_06A74F03156B-1.jpeg)

1. 該按鈕可以設定到 20 個，Omnichat 的按鈕為預設會佔掉一個名額
2. 可以選擇是否需要開啟常設功能表
3. 新增：可新增按鈕內容為開啟 URL 或是設定機器人模組，亦可以設定標籤
4. 編輯：隨時修改按鈕內容設定
5. 刪除：可刪除該按鈕
6. 預設顯示 Omnichat 訊息
