# 修改聯絡人綁定

{% endhint %}

設定教學請參考「[修改聯絡人綁定](https://docs.omnichat.ai/features/omo-she-ding/jian-li-dian-yuan#xiu-gai-lian-lao-ren-bang-ding)」。
